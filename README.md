# City Edge Landing Page — Next.js

## 🚀 طريقة التشغيل

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. تشغيل الموقع
```bash
npm run dev
```
افتح المتصفح على: http://localhost:3000

### 3. بناء للإنتاج
```bash
npm run build
npm start
```
